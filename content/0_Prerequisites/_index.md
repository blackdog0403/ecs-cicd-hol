# 시작하기

{{% notice warning %}}
본 워크샵은 AWS Oregon Region ( us-west-2 ) 기준으로 작성이 되었습니다. 실습시 유의해주시길 바랍니다.
{{% /notice %}}


워크샵을 시작하려면 다음 중 하나에 따라 다음 중 하나를 수행하십시오...

* ...[자신의 계정으로 워크샵을 직접 실습하기](self_paced/)
* ...[AWS 호스팅 이벤트 참석 (AWS 제공 해시 사용)](aws_event/)

두 설정 중 하나를 완료하였다면 다음의 링크를 통해 [**워크 스페이스를 생성**](create-a-workspace/) 합니다.
